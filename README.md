# Spectral Videography Website

This is the **canonical production website** for spectralvideography.com.

- `index.html` — main site (HTML/CSS/JS, orange + charcoal blue theme)
- `admin/index.html` — admin dashboard (view Supabase inquiries)
- `hero2.mp4` — hero background video

**Deploy:** `netlify deploy --prod --dir=. --site=83b1857d-7633-43d9-bbb1-626dba028ea0 --no-build`

Do NOT deploy the `nimbetics/` Next.js project to spectralvideography.com.
